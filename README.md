# Lost and Found Web-Based Management System for Strathmore University

A web-based platform for managing lost and found items at Strathmore University, connecting students and staff with administrators to securely report, search, and recover missing belongings.

## Technology Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP 8.x
- **Database:** MySQL (managed via MySQL Workbench)
- **IDE:** Visual Studio Code

## Project Structure
- `/backend` - Server-side logic, API endpoints, and database configuration.
- `/frontend` - Client-side UI, pages, and assets.
- `/database` - SQL schema and migration scripts.
- `/docs` - Project documentation and API contracts.
